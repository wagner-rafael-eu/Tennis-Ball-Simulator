TENNIS BALL SIMULATOR - KEYBOARD COMMANDS
==========================================

CONFIGURATION FILE
------------------
The simulator reads settings from settings.ini file located in the same directory as the executable.
You can modify the following settings:
  • DefaultForce - Initial horizontal force in Newtons (0-1000, default: 270)
  • DefaultAngle - Initial launch angle in degrees (0-90, default: 39)
  • AngleStep - Angle adjustment increment in degrees (default: 3)
  • DefaultSpin - Initial ball spin in RPM (-3000 to +9000, default: 120)
  • SpinStep - Spin adjustment increment in RPM (default: 60)
  • MinSpin - Minimum allowed spin in RPM (default: -3000)
  • MaxSpin - Maximum allowed spin in RPM (default: +9000)
  • DefaultPace - Initial visual pace in percentage (default: 200 = 2x speed)

SCREEN NAVIGATION
-----------------
BACKSPACE - Return to All Courts View (SCREEN_ALL)
C - Switch to Clay Court View (SCREEN_CLAY)
G - Switch to Grass Court View (SCREEN_GRASS)
H - Switch to Hard Court View (SCREEN_HARD)
L - Switch to Laver Cup View (SCREEN_LAVER)

SIMULATION CONTROL
------------------
SPACE - Start simulation
R - Reset simulation

CLAY & GRASS & HARD & LAVER COURT CONTROLS (SCREEN_CLAY & SCREEN_GRASS & SCREEN_HARD & SCREEN_LAVER)
-----------------------------------------------------------------------------------------------------
W key - Increase launch angle (+step per press, configurable via AngleStep in settings.ini)
        Range: 0-90°

S key - Decrease launch angle (-step per press, configurable via AngleStep in settings.ini)
        Range: 0-90°

D key - Increase horizontal force (+10N per press)
        Range: 0-1000N
           
A key - Decrease horizontal force (-10N per press)
        Range: 0-1000N

MOUSE SCROLL UP - Increase launch angle (+step per scroll, configurable via AngleStep in settings.ini)
                  Range: 0-90°
                  
MOUSE SCROLL DOWN - Decrease launch angle (-step per scroll, configurable via AngleStep in settings.ini)
                    Range: 0-90°

UP ARROW - Increase horizontal force (+10N per press)
           Range: 0-1000N
           
DOWN ARROW - Decrease horizontal force (-10N per press)
             Range: 0-1000N

CTRL + + - Add topspin (+step per press, configurable via SpinStep in settings.ini)
           Range: MinSpin to MaxSpin RPM (configurable in settings.ini)
           Effect: Ball curves downward, kicks forward on bounce

> key - Add topspin (+step per press, configurable via SpinStep in settings.ini)
        Range: MinSpin to MaxSpin RPM (configurable in settings.ini)
        Effect: Ball curves downward, kicks forward on bounce
           
CTRL + - - Add backspin (-step per press, configurable via SpinStep in settings.ini)
           Range: MinSpin to MaxSpin RPM (configurable in settings.ini)
           Effect: Ball floats upward, checks up on bounce

< key - Add backspin (-step per press, configurable via SpinStep in settings.ini)
        Range: MinSpin to MaxSpin RPM (configurable in settings.ini)
        Effect: Ball floats upward, checks up on bounce

+ (Plus key) - Increase visual pace (+10% per press)
               Range: 10%-1000% (0.1x to 10x speed)
               
- (Minus key) - Decrease visual pace (-10% per press)
                Range: 10%-1000% (0.1x to 10x speed)

MOUSE CONTROLS
--------------
LEFT CLICK on Air Resistance Combo Box - Cycle through air resistance modes:
  • Vacuum (no air resistance)
  • Sea Level (standard atmospheric conditions)
  • 1000m altitude (~10% less air resistance)
  • 2000m altitude (~22% less air resistance)

PHYSICS PARAMETERS
------------------
Force: 0-1000N → Maps to 0-50 m/s ball velocity (default: 270N, configurable)
Angle: 0° (horizontal) to 90° (vertical) (default: 39°, configurable)
Angle Step: Configurable via settings.ini (default: 3°)
Spin: MinSpin to MaxSpin RPM (default: -3000 to +9000 RPM, configurable)
Default Spin: Configurable via settings.ini (default: 120 RPM topspin)
Spin Step: Configurable via settings.ini (default: 60 RPM)
Visual Pace: 0.1x to 10x simulation speed (default: 2.0x = 200%, configurable)
Frame Rate: 120 FPS

COURT PHYSICS
-------------
Clay Court (Roland Garros):
  • Coefficient of Restitution: 0.75 (higher bounce)
  • Friction: 0.6 (slower surface)
  • Color: Orange clay
  • Ball: Yellow

Grass Court (Wimbledon):
  • Coefficient of Restitution: 0.70 (lower bounce)
  • Friction: 0.4 (faster surface)
  • Color: Green grass
  • Ball: Bright green

Hard Court (US Open):
  • Coefficient of Restitution: 0.73 (medium bounce)
  • Friction: 0.5 (medium surface)
  • Color: Blue hard court
  • Ball: Yellow

Laver Cup (Black Court):
  • Coefficient of Restitution: 0.73 (medium bounce)
  • Friction: 0.5 (medium surface)
  • Color: Black court
  • Ball: Yellow

SPIN EFFECTS
------------
Topspin (Positive RPM):
  • Magnus effect curves ball downward
  • Increases forward velocity on bounce
  • Ball dips faster and kicks forward

Backspin (Negative RPM):
  • Magnus effect lifts ball upward
  • Decreases forward velocity on bounce
  • Ball floats longer and checks up on landing

Spin decays to 70% of original value after each bounce
